Informática

A seção Informática apresenta livros essenciais para quem deseja aprofundar seus conhecimentos em programação, análise de dados e ciência da computação.

  • Clean Code - Robert C. Martin

    Um guia prático para escrever código limpo e sustentável, com dicas e técnicas para melhorar a qualidade do software.

  • Estruturas de Dados e Algoritmos em JavaScript - Loiane Groner

    Uma introdução prática e abrangente às estruturas de dados e algoritmos, explicadas com exemplos em JavaScript.

  • A Arte de Programar Computadores - Donald Knuth

    Uma obra clássica que aborda os fundamentos da ciência da computação e a lógica por trás da programação eficiente.

  • Python para Análise de Dados - Wes McKinney

    Um guia completo para usar Python em análise de dados, incluindo manipulação de tabelas, estatísticas e visualização.

  • Introdução à Ciência da Computação com Python - John V. Guttag

    Um livro ideal para iniciantes que desejam aprender ciência da computação e programação usando Python.

  • Refactoring - Martin Fowler

    Um guia prático para aprimorar a qualidade do código, com técnicas para torná-lo mais legível e sustentável, sem alterar sua funcionalidade.

  • JavaScript: O Guia Definitivo - David Flanagan

    Uma referência completa para desenvolvedores que desejam dominar JavaScript, desde os fundamentos até os recursos avançados.

  • Design Patterns - Erich Gamma e outros

    Um clássico que apresenta padrões de design reutilizáveis, ajudando a resolver problemas comuns no desenvolvimento de software.

  • Scrum: A Arte de Fazer o Dobro na Metade do Tempo - Jeff Sutherland

    Uma introdução prática ao método Scrum, com estratégias para aumentar a eficiência e produtividade nas equipes.

  • Hackers: Heroes of the Computer Revolution - Steven Levy

    Uma fascinante história dos pioneiros da computação e sua paixão pela inovação tecnológica.
